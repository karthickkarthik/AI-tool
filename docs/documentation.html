Webhook Integration

Receive real-time notifications when content is generated or processed:

POST /webhooks
Content-Type: application/json
Authorization: Bearer YOUR_API_KEY

{
  "url": "https://your-app.com/webhook",
  "events": ["content.generated", "image.processed"],
  "secret": "your_webhook_secret"
}

Webhooks

Webhooks allow you to receive real-time notifications when events occur in your AI Tools account.

Supported Events

Event Description Payload
content.generated Triggered when content generation is completed Content object with ID, title, and content
image.processed Triggered when image processing is completed Image object with ID, URL, and metadata
translation.completed Triggered when translation is completed Translation object with original and translated text

Webhook Security

All webhooks include a signature header for verification:

X-AI-Tools-Signature: sha256=abc123...
Important: Always verify webhook signatures to ensure requests are from AI Tools.

Examples

Practical examples to help you get started with our API.

JavaScript Example

const axios = require('axios');

const generateContent = async () => {
  try {
    const response = await axios.post(
      'https://api.aitools.com/v1/content/generate',
      {
        type: 'blog_post',
        topic: 'AI in Healthcare',
        length: 'medium',
        tone: 'professional'
      },
      {
        headers: {
          'Authorization': 'Bearer YOUR_API_KEY',
          'Content-Type': 'application/json'
        }
      }
    );
    
    console.log('Generated content:', response.data);
  } catch (error) {
    console.error('Error:', error.response.data);
  }
};

generateContent();

Python Example

import requests

def generate_content():
    url = 'https://api.aitools.com/v1/content/generate'
    headers = {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    }
    data = {
        'type': 'blog_post',
        'topic': 'AI in Healthcare',
        'length': 'medium',
        'tone': 'professional'
    }
    
    response = requests.post(url, json=data, headers=headers)
    
    if response.status_code == 200:
        return response.json()
    else:
        raise Exception(f'Error: {response.text}')

# Generate content
content = generate_content()
print(content)

Troubleshooting

Common issues and their solutions.

Authentication Errors

401 Unauthorized: Check that your API key is correct and included in the Authorization header.

Rate Limit Errors

429 Too Many Requests: You've exceeded your rate limit. Wait before making more requests or upgrade your plan.

Content Generation Issues

  • Ensure the topic is clear and specific
  • Check that the content type is supported
  • Verify your account has sufficient credits
